If an out-of-scope song has at least 5 song entries as alternate versions (derivatives such as remixes or covers) on VocaDB, it can be created as a [cover unifier](https://vocadb.net/T/6751) song entry. Sub-derivatives (e.g. cover of a remix) don't count for eligiblity.

For artists which has an in-scope entry, you can add the artist entry on the cover unifier song entry (e.g. using "[HACHI](https://vocadb.net/Ar/49)" for "[KICK BACK](https://vocadb.net/S/454815)"). Otherwise, add it as a custom artist. Do not create new artist entries just for crediting a cover unifier.

When no artist entry can be added, you can use [the "Temporary artist" entry](https://vocadb.net/Ar/139625) so that you can create the song entry.
